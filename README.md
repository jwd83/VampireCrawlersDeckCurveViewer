# Vampire Crawlers — Deck Curve Viewer

A BepInEx 6 IL2CPP plugin for **Vampire Crawlers** that adds a small in-game
overlay listing every card in your current run, grouped by mana cost.

## Install

1. Install [BepInEx 6 IL2CPP x64](https://builds.bepinex.dev/projects/bepinex_be) for Vampire Crawlers.
2. Launch the game once so BepInEx creates its folders.
3. Download `DeckCurveViewer.dll` from [`DeckCurveViewer/dist/`](DeckCurveViewer/dist/).
4. Drop it into `<game>/BepInEx/plugins/`.
5. Launch the game.

A `Deck Curve` button appears on the left side of the screen during a run.
Click it to toggle the deck list and mana-cost histogram.

## Build from source

```powershell
dotnet build .\DeckCurveViewer\DeckCurveViewer.csproj -c Release
```

The compiled DLL is written to `DeckCurveViewer/bin/Release/net6.0/DeckCurveViewer.dll`.

The project references game assemblies generated by BepInEx into
`BepInEx/interop/` (Pancake.dll, Sirenix.Serialization.dll) plus the standard
Unity / Il2Cpp interop assemblies. Run the game once with BepInEx installed so
those exist before building.

## Compatibility

Built against Vampire Crawlers on Unity 6000.0.62f1 with BepInEx 6.0.0-be.755.
Game updates can rename runtime types (`Nosebleed.Pancake.Models.PlayerModel`,
`CardModel`, `CardConfig`) — if the overlay reports `No active player model
found.` after an update, the strongly-typed lookup likely needs to be
re-pointed at the new symbols.
